Class: ExcecutionPattern

(protected) PoonyaExcecutionPattern(input, import_s, logger)

(protected) new ExcecutionPattern(input, import_s, logger)

Шаблон кода, все что подается сюда, будет распознаваться как код шаблонизатора:

set somevar = arrays.create();

somevar <- 'Hello ';
somevar <- 'World';

repeat(0; 2){
    > objects.get(somevar, current);
}

// По завершению выполнения шаблона будет выведено Hello World

Parameters:
Name Type Description
input String | Object Входящая строка с выражением
import_s Array.<String> Массив с нативными библиотеками для импорта
logger Object Логгер, за интерфейс нужно взять console, с функциями log, warn, error;

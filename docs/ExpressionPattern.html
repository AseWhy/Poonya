Class: ExpressionPattern

(protected) PoonyaExpressionPattern(input, import_s, logger)

(protected) new ExpressionPattern(input, import_s, logger)

Шаблон выражения, сюда можно подать на всход выражение, а на выходе получить результат выполнения этого выражения:

2 + (3 * (5 * (2 * 2)) / 3)

Пример выше выведет 22
Parameters:
Name Type Description
input String | Object Входящая строка с выражением
import_s Array.<String> Массив с нативными библиотеками для импорта
logger Object Логгер, за интерфейс нужно взять console, с функциями log, warn, error;

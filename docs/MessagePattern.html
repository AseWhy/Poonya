Class: MessagePattern

(protected) PoonyaMessagePattern(input, import_s, logger)

(protected) new MessagePattern(input, import_s, logger)

Шаблон сообщения, на вход нужно подавать код вида:

Not formatted text...
Not formatted text...
{
    // You code here
}
Not formatted text...
Not formatted text...

Т.е. код поадется туда в фигурных скобках, а все что не в них будет распознано как текст.
Parameters:
Name Type Description
input String | Object Входящая строка с выражением
import_s Array.<String> Массив с нативными библиотеками для импорта
logger Object Логгер, за интерфейс нужно взять console, с функциями log, warn, error;
